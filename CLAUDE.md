# Designer Hub

Internal design team daily reporting and performance tracking app.

## Stack
| Layer | Tech |
|-------|------|
| **Framework** | Next.js 14 (App Router), TypeScript |
| **DB/Auth/Storage** | Supabase |
| **UI** | Tailwind + shadcn/ui, Recharts |
| **Deploy** | Cloudflare Pages |

## Features
- **Submit** (`/submit`) — designers upload daily assets (images & videos)
- **Leaderboard** (`/leaderboard`) — rankings by performance scores
- **Judge** (`/judge`) — admin-only anonymous rating (1-3 stars)
- **Admin** (`/admin`) — submissions overview, ratings, performance

## Status
Live and actively used by the design team. Renamed to "Designer Hub" as part of the consolidation project.
